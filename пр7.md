## Задание 1. Работа с reqres.in

Для работы выбрал API **ReqRes** (https://reqres.in) — это публичный API для тестирования, работает с пользователями.

Чтобы API принимал запросы, нужен бесплатный ключ. Получил его на сайте reqres.in и сохранил в переменную `apiKey`. Также создал переменную `baseUrl = https://reqres.in`, чтобы не писать адрес в каждом запросе вручную. Авторизацию настроил на уровне коллекции — ключ автоматически добавляется ко всем запросам.

![Переменные коллекции](![alt text](image.png))

Создал три запроса:

**GET /api/users?page=2** — получает список пользователей. Вернул код 200 и массив с данными.

**GET /api/users/2** — получает одного пользователя по ID. Вернул код 200 и объект с полями id, email, имя.

**POST /api/users** — создаёт нового пользователя. В теле отправил имя и должность, сервер вернул код 201 и присвоил ID.

Для каждого запроса написал тесты: проверяю статус-код и что нужные поля есть в ответе.
